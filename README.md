# applet_MyCalc
简单(简陋)的安卓计算器  
可以通过app-debug.apk安装到手机上, 基本上可以代替手机自带的计算器用了😂  
好多功能是基于Java的BigDecimal搞出来的, 小数位数没有作控制, 有时会出现小数位巨多的情况  
主要是利用"数据结构"课上讲的中缀表达式转后缀表达式, 以及后缀表达式求值的方法实现的
## 特色功能
### 对数计算(指数计算当然也有了, 这里不演示了)
